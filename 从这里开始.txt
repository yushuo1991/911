⚡ 最常用的操作
═══════════════════════════════════════════════════════════════

【更新代码到服务器（三步）】

    git add .
    git commit -m "你的修改说明"
    git push

就这样！3-5分钟后自动部署完成。


【一键完成】

    git add . && git commit -m "update" && git push


【查看部署状态】

    在线查看：
    https://github.com/yushuo1991/stock-tracker/actions

    命令行：
    gh run list --repo yushuo1991/stock-tracker --limit 5
