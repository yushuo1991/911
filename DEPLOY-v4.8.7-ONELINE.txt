🚀 方法一：Git拉取重建（完整单行命令）

cd /www/wwwroot/stock-tracker && git fetch origin && git checkout -- src/app/page.tsx && git pull origin main && docker compose down && docker compose build && docker compose up -d && sleep 20 && docker ps | grep stock-tracker && curl -I http://localhost:3002 && echo "✅ v4.8.7部署完成！"

🚀 方法二：Perl热替换（完整单行命令，最快）

cd /www/wwwroot/stock-tracker && cp src/app/page.tsx src/app/page.tsx.backup-v4.8.6 && sed -i 's/z-50/z-[70]/' src/app/page.tsx && CONTAINER_ID=$(docker ps | grep stock-tracker-app | awk '{print $1}') && docker cp src/app/page.tsx $CONTAINER_ID:/app/src/app/page.tsx && docker restart $CONTAINER_ID && sleep 15 && echo "✅ v4.8.7热部署完成！"
